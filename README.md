# UserSphere Backend

This project is a Spring Boot backend application designed to demonstrate how REST APIs are built and structured.

## Current Features
- Basic REST API setup
- Test endpoint: `/test`
- Layered project structure (Controller)

## Planned Features
- User CRUD APIs (Create, Read, Update, Delete)
- Database integration using MySQL
- Service and Repository layers
- Input validation and exception handling

## Tech Stack
- Java 17
- Spring Boot
- Maven

## How to Run
1. Clone the repository
2. Open in Eclipse
3. Run UsermanagementApplication.java
4. Visit: http://localhost:8080/test
